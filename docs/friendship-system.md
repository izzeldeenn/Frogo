# نظام الصداقة والمراسلة

تم إنشاء نظام صداقة متكامل للتطبيق يسمح للمستخدمين بإرسال طلبات الصداقة وقبولها ومراسلة الأصدقاء.

## المميزات

### 🤝 نظام الصداقة
- **إرسال طلبات الصداقة**: يمكن للمستخدمين إرسال طلبات صداقة لبعضهم البعض
- **قبول/رفض الطلبات**: المستخدمون يمكنهم قبول أو رفض طلبات الصداقة الواردة
- **عرض الأصدقاء**: عرض قائمة الأصدقاء الحاليين مع معلوماتهم
- **البحث عن أصدقاء**: البحث عن مستخدمين آخرين وإضافتهم كأصدقاء
- **حالة الطلبات**: عرض الطلبات المعلقة والمقبولة والمرسلة

### 💬 نظام المراسلة
- **مراسلة فورية**: إرسال واستقبال الرسائل بين الأصدقاء
- **المحادثات الجماعية**: عرض قائمة المحادثات مع آخر رسالة
- **الرسائل غير المقروءة**: عداد الرسائل غير المقروءة لكل محادثة
- **البحث في المحادثات**: البحث عن محادثات محددة
- **حالة القراءة**: عرض ما إذا كانت الرسائل قد تمت قراءتها

## الهيكل التقني

### قاعدة البيانات
تم إنشاء الجداول التالية في Supabase:

#### `friendships`
- تخزين علاقات الصداقة النشطة
- يدعم حالة الصداقة (نشط/محظور)

#### `friendship_requests`
- تخزين طلبات الصداقة
- يدعم الحالات (معلق/مقبول/مرفوض)

#### `messages`
- تخزين الرسائل بين المستخدمين
- يدعم أنواع الرسائل (نص/صورة/ملف)
- يتتبع حالة القراءة والحذف

### المكونات

#### `FriendshipManager`
- واجهة إدارة الصداقة الرئيسية
- عرض الأصدقاء والطلبات والبحث
- قبول/رفض طلبات الصداقة

#### `MessagingSystem`
- واجهة المراسلة الكاملة
- عرض المحادثات والرسائل
- إرسال الرسائل الفورية

#### `FriendsPage`
- الصفحة الرئيسية التي تجمع بين نظامي الصداقة والمراسلة
- تبديل سلس بين الواجهتين

### الـ APIs
- `/api/friend-requests` - إرسال وجلب طلبات الصداقة
- `/api/friend-requests/[id]` - قبول/رفض طلبات الصداقة
- `/api/friends` - جلب وإزالة الأصدقاء
- `/api/messages` - إرسال وجلب الرسائل

## الأمان
- تطبيق Row Level Security (RLS) على جميع الجداول
- التحقق من صلاحيات المستخدمين للوصول إلى البيانات
- حماية البيانات الشخصية والرسائل

## الاستخدام

### الوصول إلى النظام
1. انتقل إلى `/friends` في التطبيق
2. استخدم تبويب "إدارة الصداقات" لإضافة أصدقاء جدد
3. استخدم تبويب "الرسائل" للدردشة مع الأصدقاء

### إدارة الصداقة
1. **البحث عن أصدقاء**: استخدم تبويب "البحث عن أصدقاء" للعثور على مستخدمين آخرين
2. **إرسال طلب**: انقر على "إضافة صديق" لإرسال طلب صداقة
3. **معالجة الطلبات**: في تبويب "طلبات الصداقة"، يمكنك قبول أو رفض الطلبات الواردة
4. **عرض الأصدقاء**: في تبويب "الأصدقاء"، يمكنك عرض جميع أصدقائك الحاليين

### المراسلة
1. **اختيار محادثة**: من قائمة المحادثات، اختر الصديق الذي تريد الدردشة معه
2. **إرسال رسالة**: اكتب رسالتك في مربع الإدخال واضغط على Enter أو انقر على أيقونة الإرسال
3. **البحث**: استخدم شريط البحث للعثور على محادثات محددة

## التخصيص

### إضافة المزيد من المميزات
- **الصور والملفات**: يمكن توسيع نظام الرسائل لدعم الصور والملفات
- **الإشعارات**: إضافة إشعارات فورية للرسائل وطلبات الصداقة
- **المجموعات**: إنشاء مجموعات دردشة جماعية
- **حالة التواجد**: عرض حالة تواجد الأصدقاء (متصل/غير متصل)

### تعديل الواجهة
- يمكن تخصيص الألوان والخطوط عبر Tailwind CSS
- إضافة أيقونات جديدة من Lucide React
- تعديل تخطيط المكونات حسب الحاجة

## المتطلبات
- Supabase للواجهة الخلفية وقاعدة البيانات
- Next.js للواجهة الأمامية
- React و TypeScript للمكونات
- Tailwind CSS للتصميم
- Lucide React للأيقونات
